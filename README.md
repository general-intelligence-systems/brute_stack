# brute_stack

## Start

```sh
docker compose up -d
```

## Log in

Open <https://localhost>, accept the self-signed cert, and sign in:

- **Username:** `@demo:localhost`
- **Password:** `demo`

Two chats are waiting: one with `@brute:localhost`, one with `@echo:localhost`.
